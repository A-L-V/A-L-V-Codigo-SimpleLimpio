Mas info Host Pseudoclase que utilizaremos para darle estilos a nuestro componente web (no se trata necesariamente de los estilos visuales).
Se trata de los estilos que vienen definidos por default con una etiqueta, como pueden ser display, padding y margin. Hola desde un componente 1 **:host **{estilos para el componente} Hola desde un componente 2: :host(.blue) {estilos para el componente con la clase blue} Teniendo varias instancias de un componente, si a una le agregamos una clase por ejemplo ‘blue’ Hola desde un componente 3: :host([yellow]) {estilos para el elemento que tenga el atributo yellow} También podemos darle estilos por atributo. Por ejemplo si a una instancia le agregamos el atributo ‘yellow’
Hola desde un componente 4: :host-context(article.card) {estilos} También podemos agregar cierto contexto. Por ejemplo, si tenemos una instancia del componente dentro de un article con una clase ‘card’
Hola desde un componente 5: :host([yellow]) h1 {estilos} Hacer cambios al contenido del componente
